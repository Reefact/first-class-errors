# FCE002: EmptyErrorCode

| | |
|---|---|
| **Category** | Error codes (`FirstClassErrors.ErrorCodes`) |
| **Severity** | 🔴 Error |
| **Enabled by default** | Yes |

`ErrorCode.Create` is called with an empty, whitespace, or `null` literal. That throws an `ArgumentException` at run time; this rule catches it at build time.

## Noncompliant

```csharp
public static readonly ErrorCode A = ErrorCode.Create(""); // FCE002
```

## Compliant

```csharp
public static readonly ErrorCode A = ErrorCode.Create("MONEY_TRANSFER_INVALID");
```

## Details

Only literal arguments are inspected. A code computed at run time is out of scope (see [FCE003](FCE003.md)).
