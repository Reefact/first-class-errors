# FCE015: DocumentationTitleTooGeneric

| | |
|---|---|
| **Catégorie** | Contenu de la documentation (`FirstClassErrors.DocumentationContent`) |
| **Sévérité** | 🔵 Info |
| **Activée par défaut** | Non (opt-in) |

Un `WithTitle("...")` utilise un titre qui ne décrit rien (`Error`, `Invalid value`, `Failure`, …). Un bon titre nomme la condition.

## Non conforme

```csharp
DescribeError.WithTitle("Invalid value") // FCE015
             .WithDescription("...")/* ... */;
```

## Conforme

```csharp
DescribeError.WithTitle("Temperature below absolute zero")
             .WithDescription("...")/* ... */;
```

## Activer cette règle

Cette règle est désactivée par défaut. Activez-la dans `.editorconfig` :

```ini
dotnet_diagnostic.FCE015.severity = suggestion   # ou warning / error
```
