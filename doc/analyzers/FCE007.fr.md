# FCE007: DocumentedByInvalidSignature

| | |
|---|---|
| **Catégorie** | Câblage de la documentation (`FirstClassErrors.DocumentationWiring`) |
| **Sévérité** | 🔴 Error |
| **Activée par défaut** | Oui |

La méthode référencée par `[DocumentedBy]` existe mais ne peut pas servir de factory de documentation. Une méthode de documentation doit être **statique**, **sans paramètre**, et retourner **`ErrorDocumentation`**.

## Non conforme

```csharp
[DocumentedBy(nameof(Doc))]
internal static DomainError Boom() => /* ... */;

private static string Doc() => "not a documentation"; // FCE007: wrong return type
```

## Conforme

```csharp
private static ErrorDocumentation Doc() =>
    DescribeError.WithTitle("...").WithDescription("...").WithoutRule().WithoutDiagnostic().WithExamples<DomainError>();
```

## Détails

Une cible manquante n'est pas du ressort de cette règle ; voir [FCE006](FCE006.fr.md).
