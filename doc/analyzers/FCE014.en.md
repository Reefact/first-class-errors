# FCE014: ShortMessageSameAsDetailedMessage

| | |
|---|---|
| **Category** | Documentation content (`FirstClassErrors.DocumentationContent`) |
| **Severity** | 🔵 Info |
| **Enabled by default** | Yes |

`WithPublicMessage(short, detailed)` is called with two identical literal messages. The short message is a public summary and the detailed one an optional public detail, so identical values usually signal a copy-paste.

## Noncompliant

```csharp
.WithPublicMessage("Temperature is invalid.", "Temperature is invalid."); // FCE014
```

## Compliant

```csharp
.WithPublicMessage("Temperature is invalid.", "The temperature -280 Celsius is below absolute zero.");
```
