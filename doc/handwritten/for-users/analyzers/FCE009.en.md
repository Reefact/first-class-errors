# FCE009: ErrorFactoryNotDocumented

🌍 **Languages:**  
🇬🇧 English (this file) | 🇫🇷 [Français](./FCE009.fr.md)

| | |
|---|---|
| **Category** | Documentation wiring (`FirstClassErrors.DocumentationWiring`) |
| **Severity** | 🟠 Warning |
| **Enabled by default** | Yes |

A non-private static factory that returns an `Error` in a `[ProvidesErrorsFor]` type carries no `[DocumentedBy]`. The error is never added to the generated catalog.

## Noncompliant

```csharp
[ProvidesErrorsFor(nameof(Sample))]
public static class SampleError {
    internal static DomainError Boom() => /* ... */; // FCE009: no [DocumentedBy]
}
```

## Compliant

```csharp
[DocumentedBy(nameof(BoomDocumentation))]
internal static DomainError Boom() => /* ... */;
```

## Details

Private methods are treated as helpers and left alone.
