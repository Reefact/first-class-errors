# FCE013: ExampleDoesNotCallDocumentedFactory

🌍 **Langues:**  
🇬🇧 [English](./FCE013.en.md) | 🇫🇷 Français (ce fichier)

| | |
|---|---|
| **Catégorie** | Contenu de la documentation (`FirstClassErrors.DocumentationContent`) |
| **Sévérité** | 🟠 Warning |
| **Activée par défaut** | Oui |

Un exemple passé à `WithExamples(...)` n'appelle aucune factory du type qui déclare la documentation. Les exemples existent pour exposer les vrais messages de l'erreur documentée : chacun devrait donc construire cette erreur.

## Non conforme

```csharp
// inside SampleError's documentation method
.WithExamples<DomainError>(() => Other.Build()); // FCE013: Other is a different type
```

## Conforme

```csharp
.WithExamples<DomainError>(() => Boom()); // Boom is a factory of SampleError
```

## Détails

Les formes d'exemple reconnues sont les lambdas et les groupes de méthodes. Une forme non reconnue est laissée de côté pour éviter les faux positifs.
