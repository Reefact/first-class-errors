# FCE014: ShortMessageSameAsDetailedMessage

🌍 **Langues:**  
🇬🇧 [English](./FCE014.en.md) | 🇫🇷 Français (ce fichier)

| | |
|---|---|
| **Catégorie** | Contenu de la documentation (`FirstClassErrors.DocumentationContent`) |
| **Sévérité** | 🔵 Info |
| **Activée par défaut** | Oui |

`WithPublicMessage(short, detailed)` est appelé avec deux messages littéraux identiques. Le message court est un résumé public et le détaillé un détail public optionnel : des valeurs identiques signalent en général un copier-coller.

## Non conforme

```csharp
.WithPublicMessage("Temperature is invalid.", "Temperature is invalid."); // FCE014
```

## Conforme

```csharp
.WithPublicMessage("Temperature is invalid.", "The temperature -280 Celsius is below absolute zero.");
```
