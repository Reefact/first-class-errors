Statement upload rate-limited #

STATEMENT_UPLOAD_RATE_LIMITEDStatementUploadEndpoint

Documentation

This error occurs when too many statement uploads arrive in a short window and the endpoint throttles the request. It is transient: the same request can be retried later.

Business rule: Callers must stay within the endpoint's upload rate limit.

Public summary

Statement upload rate-limited.

Public detail

Too many statement uploads were sent in a short period; please retry later.

Internal diagnostic message

Internal — not intended for external clients or HTTP responses.

The statement upload request 11111111-1111-1111-1111-111111111111 was rate-limited; retry after 30 seconds.

Diagnostics

  • The caller exceeded the allowed request rate. — origin: External — Back off and retry after the delay indicated in the message.
